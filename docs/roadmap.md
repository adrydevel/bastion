# Roadmap

Near-term, in rough order:

1. **TEE-signed attestation** — run the council inside an enclave and sign the
   proof, so it certifies where the decision ran, not only that it is consistent.
2. **Backtest harness** — replay strategies against historical oracle series
   with walk-forward evaluation.
3. **Multi-book allocation** — a portfolio layer over per-ticker councils with a
   shared risk budget.
4. **Live paper-trading** on Robinhood Chain testnet with public proof feed.

Longer-term items are tracked in issues under the `roadmap` label.
